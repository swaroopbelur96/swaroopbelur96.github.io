Intro

I'm an Internet of Things researcher from Next Tech Labs, SRM University, Chennai. I study Electronics and Communication Engineering here and I'm in my 7th semester. Having worked on many projects, I feel acquainted with common development boards like Arduino, Raspberry Pi, Intel Edison as well as ESP8266 and various ESP8266 development boards like NodeMCU. By the way, check out my awesome work.

I'm from the city of Bangalore in Karnataka but I've travelled and lived in various countries including Indonesia, the Kingdom of Saudi Arabia and the United Kindgom.

Work

  • As part of a team of three, won a hackathon hosted by IESA(Indian Electronics and Semiconductor Association). We built an industrial IoT system using Intel Edison and various sensors that collects various sensor information like temperature, pressure, vibration etc. and makes it available for constant observation online. We also implemented a machine learning algorithm using Scikit-Learn that takes in these sensor values and tells if the the machine being observed is in proper order. Finally, we also built an Augmented Reality app made using Unity with Vuforia that when pointed to a sticker on an industrial machine, gives live sensor data.
  • Awarded 3rd in the Project Expo at SRM University: Project was an AI based driver assistance system that stops the car automatically at STOP signs and Traffic Signals when red. It also alerts the driver about various traffic signs as well as pedestrians. This was done using Haar-Cascades in OpenCV running on a Raspberry Pi. The system was implemented on a small Remote Controlled Car.
  • Entrance System at Next Tech Labs using RFID and electromagnetic lock: An RC522 module was used with an Arduino Uno which controlled an electromagnetic lock through a relay.
  • Automation of Next Tech Lab: All lights and fans are now controlled through a webpage over WiFi. The project was done by controlling relays through an Arduino Mega. The Mega was serially given information from an ESP8266.
  • Co-authored a research paper titled "Two Dimensional Object Positioning System Based On Infrared Technology" M Meenakshi Sivapriya, Chirag Parmar M, and Swaroop Sudhanva Belur; SRM Research Day 2017
  • Was invited by the Institution of Electronics and Telecommunication Engineers(IETE) to give a talk on Internet of Things and its advancements.
  • Plan Moisture Sensor: A Darlington Pair setup of BC547 transistors was used to detect the presence of water in the soil. The project was awarded 'Best Project' by ABET(Accreditation Board for Engineering and Technology)
  • Intruder Detector using IoT: The project consisted of an HC SR04 ultrasonic sensor and a NodeMCU. When the distance to some obstruction was sensed to be below a certain threshold, the NodeMCU made an HTTP request to IFTTT which gave a notification on a mobile phone.
  • Digital Volume Unit Meter: Used an LM3915 chip to visually indicate the volume of sound sensed by a microphone.
